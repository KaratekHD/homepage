Kircle - Rechnen mit Kreisen

Ein kleines Java Programm um mit Kreisen zu rechnen. Das Programm ist komplett FLOSS und Freie Software (Was ist das?). Jeder darf es also weitergeben, benutzen, ändern und studieren wie er will, solange er sich an die Lizenz (siehe unten) hält! Der Quellcode steht auf GitHub zur Verfügung. Sollte es Probleme geben, kann dort auch eine “Issue”, eine Art Forenthread, geöffnet werden, ich gebe mir dann Mühe zu helfen.
