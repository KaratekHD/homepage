Minecraft Crossover

So, what is Crossover? Crossover is a set of pre configured plugins and server settings that allow us to deploy Minecraft Servers in no time. Itâ€™s fast, scallable, has an anti-cheat system and grief protection.

Server
Minecraft Crossover uses the PaperMC server software. This allows us to build scallable, high performance servers with an easy to use API.
Plugins
We use a ton of plugins in order to build Crossover. Even though we provide a list of them, we highly recommend reaching out to us if you wanna build something based on Crossover, as most of these plugins require a ton of configuration in order to get it working just right.
Large networks
Crossover has support for large networks and can (in theory) handly hundreds of players at the same time. Due to how a Minecraft Server is structured, this requires a setup using a BungeeCord Proxy, which splits players between multiple subservers.
Cross platform
We use a system based on GeyserMC and Floodgate to allow cross platform multiplayer with PC users (Java Edition) and the Bedrock Edition (Playstation, Xbox One, Nintendo Switch, Android, iOS and Windows 10). It's not yet perfect, but works most of the time.
