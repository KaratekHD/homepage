Download Packages for GNU/Linux based Operating Systems

You can install most of our software with the package manager of your distribution.

Debian

Package manager installs are only supported on the x64 architecture. This guide describes how to use a package manager to install various Karatek Software (and some other cool stuff) on Debian. We offer packages for the latest version of Debian testing.

Cournel

Installing our repositorys on Debain Cournel is really easy. First of all, open a terminal and make sure that you are logged in as root:

sudo -s

Next step is importing our key into APT:

wget -q -O - http://download.karatek.net/repo/KEY.gpg | apt-key add -

Then you can add the repositorys to APT. For propritary software (non-free):

echo "deb https://download.karatek.net/repo/debian/non-free/cournel/ /" > /etc/apt/sources.list.d/karatek-non-free.list

After that, update your package list:

apt update

And that’s it! You should now be able to install some software, throgh Synaptic (recommended), Gnome-Software or using apt-get install <package>.

As a small example, we are gonna give you instructions about how to install Teamspeak3, contained in the non-free repository:

sudo apt-get install teamspeak3

Ubuntu

Package manager installs are only supported on the x64 architecture. This guide describes how to use a package manager to install various Karatek Software (and some other cool stuff) on Ubuntu.

Warning: Never ever use a repository for another version of your distribution! If you don’t know what version of Ubuntu you are using, fire up a terminal and run lsb_release -d

Next step is importing our key into APT:

And that’s it! You should now be able to install some software, throgh Synaptic (recommended), Gnome/Ubuntu-Software Center or using apt-get install <package>.
